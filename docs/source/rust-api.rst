Rust API (Internal)
===================

Atompack's storage engine is implemented in Rust. The Rust crate API docs are generated with
``cargo doc`` (rustdoc) and published alongside these Sphinx docs.

- `atompack (Rust crate) API docs <../rustdoc/atompack/index.html>`_
- `atompack-py / atompack_rs (Rust bindings crate) API docs <../rustdoc/atompack_rs/index.html>`_

Notes
-----

- The Rust docs are intended for contributors and debugging internals.
- The user-facing API is the Python module documented in :doc:`autoapi/index`.
